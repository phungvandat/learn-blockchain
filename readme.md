# LEARN BLOCKCHAIN

> This source referenced from [Blockchain in Go of the author Ivan Kuznetsov](https://github.com/Jeiwan/blockchain_go)

## HOW TO RUN

- Install tool:

```
make install
```

- Add blocks to blockchain:

```
learn_blockchain add "block 1" "block 2"
```

- Print all blocks:

```
learn_blockchain print
```
